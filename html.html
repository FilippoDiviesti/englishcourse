HTML (HyperText Markup Language)

This language defines all rules which describe
all mechanisms of representation

The structure of an HTML page is divided into 4 parts:

- DOCTYPE
- HTML
- HEAD
- BODY

The head tag is useful to specify all control and service information's

In the body tag we can put all web objects we want to view into our web page

This tag makes your website more dynamic by connecting multiple pages
